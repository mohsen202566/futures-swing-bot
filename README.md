# Crypto 4H Trend Pullback Toobit Bot

نسخه ریشه‌ای برای GitHub/VPS. فایل‌ها مستقیم در ریشه ریپو قرار می‌گیرند و با `python main.py` اجرا می‌شوند.

## هدف نسخه جدید

این نسخه منطق 1H قبلی را به ربات 4H تبدیل کرده است:

- تایم‌فریم اصلی تحلیل، سیگنال، استاپ و تی‌پی: `4H`
- فیلتر فشار کلی بازار: `1D`
- تریگر ورود دقیق‌تر: `1H`
- استاپ دیگر 1H نیست؛ `SL = ساختار 4H + بافر ATR`
- TP با RR ثابت `1.5R` چیده می‌شود.
- سیگنال قوی فقط برچسب قدرت می‌گیرد؛ RR در نسخه اول ثابت می‌ماند.
- ورود دیر، رنج، EMA صاف، قیمت بین EMA50/EMA200، و کندل‌های کشیده آخر روند رد می‌شوند.
- هوش مصنوعی، DCA، Martingale، trailing، چند TP و فیلتر حمایت/مقاومت نداریم.

## قوانین قفل‌شده

- تحلیل فقط از OKX.
- اجرای واقعی فقط روی Toobit Futures.
- مارجین همیشه Isolated.
- حداکثر پیش‌فرض ۵۰ ارز در هر چرخه.
- نمادها قبل از تحلیل با Toobit چک می‌شوند؛ اگر نماد در Toobit نبود، همان ارز رد می‌شود.
- اگر OKX برای نماد/کندل خطا بدهد، همان ارز رد می‌شود و کل ربات نمی‌خوابد.
- اگر 1D خلاف 4H باشد، معامله رد می‌شود.
- اگر 1D خنثی باشد و 4H قوی باشد، معامله مجاز است.
- امتیاز از ۱۰۰ است.
- زیر ۷۵: بدون سیگنال.
- ۷۵ تا ۸۴: سیگنال معمولی با RR = 1.5.
- ۸۵ به بالا: سیگنال قوی با RR = 1.5.
- کارمزد رفت‌وبرگشت ثابت پیش‌فرض: 0.05 USDT.
- اگر ترید خاموش باشد یا اسلات پر باشد، سیگنال فقط Normal ثبت می‌شود.
- اگر ترید روشن و اسلات آزاد باشد، Real روی Toobit باز می‌شود.
- اگر اسلات پر باشد، بعد از ۷۰ ثانیه Toobit چک می‌شود؛ اگر پوزیشن هنوز هست هیچ، اگر نیست اسلات آزاد می‌شود.
- مانیتورینگ TP/SL، PnL خام، PnL خالص، MFE/MAE، مدت و وضعیت Real/Normal را در SQLite ثبت می‌کند.

## منطق نهایی سیگنال

### لانگ

```text
1D خلاف جهت نباشد
4H روند صعودی واضح داشته باشد
Close_4H > EMA200_4H
EMA50_4H > EMA200_4H
EMA50_4H شیب مثبت
+DI_4H > -DI_4H
بازار رنج نباشد
پولبک سالم 4H به EMA20/EMA50 انجام شده باشد
ورود دیر نباشد
1H کندل تایید صعودی بدهد
SL زیر ساختار معتبر 4H با بافر ATR
TP = Entry + 1.5R
```

### شورت

```text
1D خلاف جهت نباشد
4H روند نزولی واضح داشته باشد
Close_4H < EMA200_4H
EMA50_4H < EMA200_4H
EMA50_4H شیب منفی
-DI_4H > +DI_4H
بازار رنج نباشد
پولبک سالم 4H به EMA20/EMA50 انجام شده باشد
ورود دیر نباشد
1H کندل تایید نزولی بدهد
SL بالای ساختار معتبر 4H با بافر ATR
TP = Entry - 1.5R
```

## فایل‌های اصلی

- `bot.py` / `main.py` — اجرای ربات و پنل تلگرام.
- `config.py` — تنظیمات از Environment Variables.
- `strategy_4h_simple.py` — منطق اصلی 4H.
- `toobit_client.py` — اجرای واقعی Toobit؛ ساختار قبلی حفظ شده.
- `okx_data.py` — کندل و قیمت OKX با retry.
- `monitor.py` / `monitoring_result_4h.py` — مانیتور TP/SL و نتیجه ریپلای.
- `storage.py` — SQLite، آمار، سیگنال‌ها و تنظیمات.
- `telegram_ui.py` — پنل، سیگنال، نتیجه و قوانین.

## اصلاح مهم Toobit/OKX

- تحلیل، کندل، فیلتر و ساخت سیگنال فقط از OKX انجام می‌شود.
- در اسکن عادی هیچ اعتبارسنجی نماد با Toobit انجام نمی‌شود تا خطای 429 ندهد.
- Toobit فقط وقتی استفاده می‌شود که ترید واقعی روشن باشد و سیگنال واجد شرایط بخواهد Real باز شود.
- پنل ممکن است فقط برای نمایش مارجین Toobit یک درخواست حساب بزند.
